## 规则

1. 以 commit 时间结算；
2. 最先独立完成并提交通过的人得胜，其他人做两分钟平板支撑；
3. 如果讨论得出答案，最先提出正确思路并且提交通过的人得胜，其他人做两分钟平板支撑；
4. 如果最先提出思路的人没有第一个提交通过，所有人做一分钟平板支撑；
